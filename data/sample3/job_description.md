# Job Description — Junior Software Engineer

We are hiring a Junior Software Engineer to join a product team and build customer-facing features.

## Responsibilities
- Implement features with guidance and learn team practices
- Write maintainable code and tests
- Debug issues and communicate progress clearly

## Requirements
- 0–2 years experience (or equivalent projects)
- Comfortable with one language (Python/Java/TypeScript)
- Basic understanding of APIs and databases
- Willingness to learn and take feedback

## Nice-to-have
- Any internship or project experience
- Familiarity with testing and CI
